## 📌 Project Overview
The **Online Learning Performance Analysis Dashboard** is an interactive data visualization project built using Microsoft Excel.  
It analyzes student performance in online learning environments by examining engagement levels, academic scores, risk categories, and dropout trends.

The dashboard helps in identifying patterns and provides insights to support **data-driven decision-making** for improving student outcomes.

## 🎯 Objectives
- Analyze student engagement and performance
- Identify high-risk and dropout students
- Understand the impact of engagement on academic results
- Compare performance across regions and education levels
- Provide actionable insights for improving learning outcomes

---

## 📂 Dataset Description
The dataset contains the following columns:

- `id_student` – Unique student ID  
- `gender` – Student gender  
- `region` – Geographic location  
- `highest_education` – Education level  
- `total_clicks` – Student interaction level  
- `avg_score` – Average score  
- `engagement_level` – Low / Medium / High  
- `performance_level` – Performance category  
- `risk_level` – Risk classification  
- `pass_flag` – Pass (1) / Fail (0)  
- `dropout_flag` – Dropout (1) / Not (0)

## 📈 Key Insights
- Higher engagement leads to better academic performance  
- Low engagement is strongly linked to higher dropout rates  
- Certain regions show variations in average performance  
- High-risk students can be identified early using engagement and score patterns  
- Students with more interaction (clicks) tend to perform better
